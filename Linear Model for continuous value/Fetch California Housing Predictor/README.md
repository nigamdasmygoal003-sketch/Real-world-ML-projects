# 🏠 House Price Prediction App

A machine learning web application built using **scikit-learn** and **Streamlit** to predict house prices based on user input features.

---

## 🚀 Features

- Predict house prices using ML model
- Built with Ridge Regression
- Interactive UI using Streamlit
- Scaled inputs using StandardScaler
- Real-time prediction

---

## 🧠 Tech Stack

- Python
- scikit-learn
- Streamlit
- NumPy, Pandas
- Joblib

---

## 📊 Dataset

- California Housing Dataset (from sklearn)
