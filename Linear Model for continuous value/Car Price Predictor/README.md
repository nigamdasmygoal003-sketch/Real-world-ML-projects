# 🚗 Car Price Predictor (Machine Learning App)

A complete end-to-end Machine Learning project that predicts car prices based on vehicle features.

Built using **scikit-learn** and deployed as a **desktop application using CustomTkinter**.

---

## 🚀 Features

- Predict car price based on multiple inputs
- Handles real-world messy data (strings, missing values)
- Clean and modern desktop UI
- Supports categorical + numerical features
- Uses RandomForest for accurate predictions

---

## 📊 Dataset

- Rows: 19,000+
- Features: 17 input features
- Target: `Price`

### Key Features:
- Manufacturer
- Model
- Year
- Engine Volume
- Mileage
- Fuel Type
- Transmission
- Condition
- Airbags
- And more...

---

## 🧠 Machine Learning Pipeline

- Data Cleaning (string → numeric conversion)
- Missing value handling (SimpleImputer)
- Feature Scaling (StandardScaler)
- Encoding (OneHotEncoder)
- Model Training (RandomForestRegressor)
- Model Evaluation (R² Score, RMSE)
