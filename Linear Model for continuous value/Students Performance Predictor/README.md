# 🎓 Student Performance Prediction App

A Machine Learning desktop application that predicts student performance based on study habits and lifestyle factors.

Built using **scikit-learn** and a modern UI with **CustomTkinter**.

---

## 🚀 Features

- Predict student performance score (Performance Index)
- Clean and modern desktop UI
- Real-time prediction based on user inputs
- Handles both numerical and categorical data
- Model trained using regression algorithms

---

## 🧠 Problem Statement

Given student-related features like:

- Hours Studied
- Previous Scores
- Extracurricular Activities
- Sleep Hours
- Sample Question Papers Practiced

👉 Predict the **Performance Index**

---

## 📊 Dataset Info

- Rows: 10,000
- Features: 5 input + 1 target
- Target: `Performance Index`

### Columns:
- Hours Studied (int)
- Previous Scores (int)
- Extracurricular Activities (categorical: Yes/No)
- Sleep Hours (int)
- Sample Question Papers Practiced (int)
- Performance Index (float)

---

## 🧠 ML Approach

- Data preprocessing (encoding categorical feature)
- Feature scaling
- Regression model (Linear / Ridge / Lasso)
- Model evaluation using metrics like R² score
