# 🚀 Real-World Machine Learning Projects

This repository contains a collection of **real-world Machine Learning projects** organized by model type.
Each project includes data preprocessing, model training, evaluation, and a deployable application (Streamlit).

## ⚙️ How to Run Any Project

```bash
# Clone the repository
git clone https://github.com/nigamdasmygoal003-sketch/Real-world-ML-projects.git

# Navigate to any project folder
cd "Real-world-ML-projects/<folder-name>/<project-name>"

# Install dependencies
pip install -r requirements.txt

# Run the app
python app.py
```
